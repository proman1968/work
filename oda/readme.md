# ODA — визуальный фреймворк

Веб-фреймворк на базе Web Components и реактивной системы (Reactor).

## Состав

- `oda.js` — ядро фреймворка: регистрация компонентов, директивы (`~if`, `~for`, `~class`), рендеринг
- `reactor.js` — реактивная система: прокси-объекты, коллектор зависимостей, события `changed`
- `utils.js` — утилиты (мёртвый файл, кандидат на удаление)
- `components/` — библиотека компонентов (button, input, editor, markdown-viewer и др.)
- `tools/` — инструменты (icon-set, chart, isie и др.)
- `benchmarks/` — тесты производительности

## Принцип

Reactor создаёт прокси над целевым объектом. При чтении свойства collector зависимостей запоминает связь «свойство → подписчик». При записи — инвалидация зависимостей и перерисовка.

Компоненты объявляются через `ODA(prototype)` — это аналог customElements.define с интеграцией Reactor.
