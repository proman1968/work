# $server/ — метапапка корня

Метапапка корневого хранилища WORK. Содержит системные типы, шаблоны и библиотеки, из которых собирается вся система.

## Структура

- `data.js` — конфигурация корневого хранилища
- `$folder/` — системный типизатор: все базовые типы (`$file`, `$storage`, `$handler`), handlers, lib

## Роль

`$server` — это метапапка (имя начинается с `$`) корневого `$storage`. Все элементы с `$` являются системными и управляются только администраторами.

Через наследование (`~` — tilde) содержимое `$server/$folder/` доступно каждому хранилищу в системе.
