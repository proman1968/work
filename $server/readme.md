# $server/ — метапапка корня

Метапапка корневого класса WORK. Содержит системные типы, шаблоны и библиотеки, из которых собирается вся система.

## Структура

- `class.js` — конфигурация корневого класса
- `$folder/` — системный типизатор: все базовые типы (`$file`, `$class`, `$handler`), handlers, lib

## Роль

`$server` — это метапапка (имя начинается с `$`) корневого `$class`. Все элементы с `$` являются системными и управляются только администраторами.

Через наследование (`~` — tilde) содержимое `$server/$folder/` доступно каждому классу в системе.
