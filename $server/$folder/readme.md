# $server/$folder/ — системный типизатор

Корневой типизатор платформы. Содержит определения всех базовых типов и их наследников. Доступен каждому классу через наследование (`~` — tilde).

## Структура

- `class.js` — конфигурация базового `$folder`
- `$file/` — тип файлов и его подтипы по расширениям (`$ai`, `$txt`, `$md`, `$prompt` и др.)
- `$class/` — тип класс и его подтипы (`$class`, `$device`, `$structure`)
- `$handler/` — тип обработчиков (`$task`, `$trigger`)
- `handlers/` — серверные и клиентские обработчики (`methods/`, `pages/`)
- `lib/` — библиотеки прикладных компонентов (`chat-item`, `tree`, `explorer`, `email` и др.)

## Принцип

Любая папка в системе наследует структуру отсюда через `collect_tilde()`. Например:
- `~/handlers/pages/form/` — форма элемента
- `~/lib/chat-item/` — компонент карточки
- `~/triggers/on_save/` — триггер сохранения
