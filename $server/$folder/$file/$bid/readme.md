# $bid — заявка

## 1. Что это

Тип файла `.bid` — базовая заявка на покупку позиции витрины (`MARKET`). JSON с покупателем, снимком товара и ответами формы.

## 2. Зачем это нужно

Единый артефакт «хочу купить» для любых товаров витрины. Исполнение (provision и т.п.) подключается отдельно, не в типе файла.

## 3. Как это работает

1. Витрина MARKET собирает `input` по `FIELDS` формы товара.
2. Клиент сохраняет `{timestamp}.{uid}.bid` через `product.save_file(new File(…))` (как чат / `task.ai`).
3. Тело: `status`, `role: 'USER'`, `buyer`, `target`, `product`, `input`.
4. Preview показывает JSON заявки.

## 4. Из чего это состоит

- `class.js` — схема FIELDS заявки, label/icon
- `triggers/on_save` — no-op в MVP
- `handlers/preview` — просмотр заявки
