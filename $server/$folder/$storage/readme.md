# $server/$folder/$storage/ — тип хранилищ

Прототип всех типизированных хранилищ. Класс `$storage extends $folder` — папка с метапапкой и `data.js`.

## Структура

- `data.js` — конфигурация: `$public.form = 'chat'`
- `$class/` — тип «класс» (метаданные)
- `$device/` — тип «устройство»
- `$structure/` — тип «структура»
- `handlers/` — специфичные для хранилища обработчики

## Методы `$storage`

- `load()` — загрузка и merge `data.js` из цепочки наследования
- `import()` — импорт `data.js` как ES-модуля
- `save()` — сохранение `data.js` (только разница с точкой наследования)
- `logs()` — универсальный доступ к логам (mode: folder/bodies/index/files)
- `read_log_bodies()` — тела записей логов за период
- `task_reply()` — продолжение диалога в task.ai
- `read_secret()` / `save_secret()` — секреты модулей в `#system/`

## Метапапка

Метапапка (имя на `$`) — основная рабочая область хранилища. Содержит `data.js`, пользовательские файлы и папки. Для элементов внутри метапапки: `$storage == $owner`.
