# $server/$folder/lib/ — библиотеки компонентов

Многоразовые библиотечные компоненты, доступные каждому элементу через `~/lib/`.

## Структура

- `chat-item/` — карточка файла/сообщения в чате
- `tree/` — дерево элементов (item-tree); опциональный `hideReadme`
- `node/` — узел дерева (item-node); «?» открывает контекстный `readme.md` уровня
- `node-explorer/` — навигатор по системе
- `explorer/` — главная страница-исследователь
- `file-viewer/` — просмотрщик файлов
- `icon/` — компонент иконки
- `menu/` — контекстное меню; скрывает `readme.md` в handlers (`hide-readme`), «?» для корневого readme
- `dialogs/` — диалоговые окна
- `pack/` — работа с пакетами файлов
- `security/` — компоненты безопасности
- `tools/` — вспомогательные инструменты
- `email/` — библиотека работы с почтой

## Использование

Компоненты подключаются через `imports` в handler'ах:
- `~/lib//chat-item` — карточка чата
- `~/lib//tree` — дерево элементов
- `~/lib//node` — узел дерева

Каждая папка содержит `class.js` с ODA-компонентом. `readme.md` на уровне — контекстная документация: в меню handlers убирается из списка (кнопка «?»), в explorer остаётся видимым.
