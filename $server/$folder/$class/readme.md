# $server/$folder/$class/ — тип класс

Прототип всех типизированных класс. Класс `$class extends $folder` — папка с метапапкой и `class.js`.

## Структура

- `class.js` — конфигурация: `$public.form = 'chat'`
- `$class/` — тип «класс» (метаданные)
- `$device/` — тип «устройство»
- `$structure/` — тип «структура»
- `handlers/` — специфичные для класса обработчики

## Методы `$class`

- `load()` — загрузка и merge `class.js` из цепочки наследования
- `import()` — импорт `class.js` как ES-модуля
- `save()` — сохранение `class.js` (только разница с точкой наследования)
- `logs()` — универсальный доступ к логам (mode: folder/bodies/index/files)
- `read_log_bodies()` — тела записей логов за период
- `task_reply()` — продолжение диалога в task.ai
- `read_secret()` / `save_secret()` — секреты модулей в `#system/`

## Метапапка

Метапапка (имя на `$`) — основная рабочая область класса. Содержит `class.js`, пользовательские файлы и папки. Для элементов внутри метапапки: `$class == $owner`.
